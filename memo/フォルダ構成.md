## 補足

| ディレクトリ/ファイル | 役割 |
|---|---|
| `main/link/linkBridge.js` | Ableton Link はネイティブモジュール（abletonlink 等）をメインプロセスで動かし、IPC で BPM/ビート位置をレンダラーへ送る |
| `renderer/audio/BpmSync.js` | Link から受け取った BPM・ビート位置をもとに、グリッドの再生タイミングを Web Audio API の `AudioContext.currentTime` に合わせる |
| `renderer/audio/TimeStretcher.js` | `soundtouchjs ^0.2.3` を使い、BPM変化に応じてサンプル長をリアルタイムに伸縮する |
| `renderer/audio/FadeManager.js` | タイムストレッチ後も音が重なる場合、次の音の直前でフェードアウトを挿入する |
| `renderer/midi/MidiMapping.js` | 再生/停止・エフェクト切り替え・x2モード等をパッドやスライダーに割り当てる設定を保持する |
| `assets/audio/` + `assets/json/` | ファイル名（連番など）で対応付け。メインプロセスが `fs.readdir` で一覧を返し、レンダラーの `FileSelector.js` でUI選択する |
